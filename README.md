# DevOps_ITA
## Docker
### Starting Docker
```docker-compose up```
### Stopping Docker
```docker-compose down```
## Vagrant
### Starting Vagrant
```vagrant up```
### Stopping Vagrant
```vagrant halt```
## Terraform
### Starting Terraform
<code>terraform init<br>
terraform validate<br>
terraform apply</code>
### Stopping Terraform
```terraform destroy```
